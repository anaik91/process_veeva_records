# Process VEEVA Records

### Pre-Requistes
* Install python3
* Access TO VEEVA

### Install Python Libraries
Install requests
```
python3 -m pip install requests
```

> Default Chunk Size is 100 
> To Override 
> On Windows `set CHUNK_SIZE=200`
> On Linux `export CHUNK_SIZE=200`

### Running Code 
```
python3 main.py
```
